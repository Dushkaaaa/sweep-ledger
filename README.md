# 🧹 Trackora

A modern web application for cleaning companies to manage employees, track working hours, calculate salaries, and generate professional reports.

Built with **Next.js**, **TypeScript**, and **Supabase**.

---

## ✨ Features

### 🔐 Authentication

- User registration
- Secure login
- Authentication powered by Supabase

---

### 👥 Employee Management

The company owner can:

- Add employees
- Edit employee information
- Remove employees
- Manage all employee records from one dashboard

---

### ⏱️ Working Hours Tracking

- Record daily working hours
- Track employee attendance
- View complete work history
- Organize work records efficiently

---

### 💰 Salary Calculation

Automatically calculate employee salaries based on recorded working hours.

Features include:

- Hourly wage calculations
- Salary overview
- Accurate payroll management

---

### 📄 PDF Report Generation

Generate professional PDF documents, including:

- Salary reports
- Weekly reports
- Monthly reports

Perfect for payroll and accounting purposes.

---

### 🌍 Multi-language Support

The application supports four languages:

- 🇵🇱 Polish
- 🇺🇦 Ukrainian
- 🇩🇪 German
- 🇬🇧 English

Users can switch languages instantly.

---

### 🏢 Company Branding

Customize the application by uploading your own company logo.

Sweep Ledger was created to simplify workforce management for cleaning companies by providing an intuitive interface for employee administration, payroll calculation, and report generation.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Dushkaaaa/sweep-ledger.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
