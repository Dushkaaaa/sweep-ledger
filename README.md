# Cleaning Tracker

Чистий стартовий проєкт на Next.js, TypeScript і App Router без шаблонного демо-контенту.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Structure

- `src/app/layout.tsx` — кореневий layout
- `src/app/page.tsx` — головна сторінка
- `src/app/globals.css` — глобальні стилі
